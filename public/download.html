DJ Booth Pi Installer

Download the DJ Booth app files to install on your Raspberry Pi.

Download for Pi

After downloading, run on your Pi:

  1. Open the Pi terminal
  2. Create the app folder: 
    mkdir -p ~/djbooth ~/data
  3. Extract the files: 
    cd ~/djbooth
tar xzf ~/Downloads/djbooth-pi-install.tar.gz
  4. Install dependencies: 
    npm install
  5. Start the server: 
    NODE_ENV=production DB_PATH=~/data/djbooth.db node server/index.js
  6. Open Chromium to http://localhost:3001
